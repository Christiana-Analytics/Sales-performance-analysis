# Sales-performance-analysis
An end to end analysis of sales data to identify revenue leaks and growth opportunities.
Business Problem; The business was experiencing inconsistent profit margin growth in comparision to their revenue and needed to understand where the problem was coming from.
Dataset; Source: Sales transaction Data (Excel), Size: 3000+ rows, Issues: Wrong Calculations, Inconsistent date formats.
Tool Used; Excel/Power Query
Process; (1). Cleaned and Standardized the data using Power Query. (2). Created a data table for seasonal and Monthly Analysis. (3). Built Data Model Relationshio to avoild double counting. (4). Analysed the table using Pivot tables. (5). Designed 2 Dashboards(Executive and Performance Analysis Dashboards). (6). Created an Action and Insights page where I carefully wrote the datastory of each chart to connection with the others while giving sure business recommendations to salvage every issue i detected.
